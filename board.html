Board Access

ATK-IMX6U

家里 Linux 板子的远程访问入口。接路由器时优先用 192.168.3.50，插电脑直连时继续用 192.168.10.2。

Tailscale 自用维护

适合自己调试硬件、SSH 登录、查看 IO 控制台。路由器/Wi-Fi 场景优先使用家里主网段地址。

  • 路由器/Wi-Fi192.168.3.50:80
  • 电脑直连192.168.10.2:80
  • SSHroot@192.168.3.50 或 192.168.10.2
打开路由器入口 打开直连入口

路由器入口需要板子接入家里路由器或 Wi-Fi；直连入口需要板子网线接在这台 Windows 电脑上。

Cloudflare 分享访问

适合临时给别人浏览器访问。代理会优先连接 192.168.3.50，失败后自动回退到 192.168.10.2。

  • 公网入口trycloudflare.com
  • 代理Windows 127.0.0.1:8090
  • 优先目标192.168.3.50
  • 回退目标192.168.10.2
打开 Cloudflare 入口

这是临时 quick tunnel 地址，重启 cloudflared 后可能变化；密码请通过单独渠道提供。

SSH 命令

旧版 Dropbear 只提供 ssh-rsa，需要在新版本 OpenSSH 中显式允许该算法。

ssh -o HostKeyAlgorithms=+ssh-rsa -o PubkeyAcceptedAlgorithms=+ssh-rsa root@192.168.3.50

使用顺序

  1. 板子接路由器或 Wi-Fi：优先打开 192.168.3.50。
  2. 板子插电脑直连：打开 192.168.10.2。
  3. 分享访问：确认 Windows 上 Node 登录代理和 cloudflared 都在运行，再发送 Cloudflare 入口。
  4. 不要把 SSH 密码或 Cloudflare 访问密码写在公开网页里。
